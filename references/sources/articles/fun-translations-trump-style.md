## 核心特征

### 1. 极简与重复
- 短小简单句子结构，传播效率极高
- 连续重复关键词或短语2-3次强化语气
- 更接近口述民间传统而非正式政治修辞

### 2. 情感与夸张
- 几乎对所有事物使用最高级
- 直接的情感诉求，而非复杂逻辑论证

### 3. 模糊性与权威感
- "很多人都在说""大家都知道"——暗示权威但不提供具体来源
- 跳跃性话题，通过联想从一个话题到另一个话题

## 标志性表达

| 表达 | 功能 |
|------|------|
| **Believe me** | 呼吁信任，频繁过渡标记 |
| **Tremendous** | 通用正面最高级 |
| **Bigly / Big league** | 强调语气（发音有争议） |
| **Many people are saying** | 无归属的权威暗示 |
| **Like nobody's ever seen** | 史无前例比较 |
| **It's a disaster** | 负面评估，频繁重复 |
| **Make X Great Again** | 核心口号结构 |
| **You're fired** | 《学徒》标志用语 |
| **The best / The worst** | 极端二元评价 |
| **Beautiful / Horrible** | 情感化形容 |

## 句式习惯
- 简单直接句式，避免复杂从句
- "and by the way"和"believe me"作为话题转变信号
- 口语化风格，对特定受众极其有效且令人难忘

## 文化影响
- 《周六夜现场》模仿曾获艾美奖
- 2015年以来关于其沟通风格的学术研究大幅增加
